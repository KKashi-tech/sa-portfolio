# Изменение типа комнаты

**Когда:** как пользователь, я хочу изменять тип комнаты, чтобы устройства и сценарии отображались корректно.  

**Примечания:**  
- Тип выбирается из предопределённого списка  
- Один тип за операцию изменения  

**Критерии приёмки:**  
- Пользователь может изменить тип комнаты  
- Сообщение, если выбран тот же тип  

**Сценарий 1: Успешное изменение типа**  
- **Дано:** пользователь в приложении, видит список комнат  
- **Когда:** выбирает «Изменить тип» → выбирает новый тип → подтверждает  
- **Тогда:** тип комнаты обновлён  
- **Сообщение:**  
> Тип комнаты изменён.


**Сценарий 2: Тип не изменён (тот же тип)**  
- **Дано:** пользователь на экране изменения типа  
- **Когда:** выбирает тот же тип → подтверждает  
- **Тогда:** сообщение:  Тип комнаты не изменён — выбран тот же тип.
- Тип комнаты не изменяется
