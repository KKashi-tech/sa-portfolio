## Удаление устройства из комнаты

**Как пользователь**,  
я хочу удалять устройство из комнаты,  
чтобы оно не отображалось среди устройств, которыми я больше не управляю в этом помещении.

### Примечания
- Удаление устройства из комнаты не удаляет его из дома.
- Устройство можно добавить в другую комнату после удаления.

### Критерии приёмки
- Авторизованный пользователь может удалить устройство из комнаты.
- Если удаление невозможно (пропало соединение с сетью или что-то еще), отображается ошибка.

### Сценарий 1: Авторизованный пользователь может удалить устройство из комнаты

**Дано**
- авторизованный пользователь находится на экране комнаты
- в списке устройств выбрано нужное устройство

**Когда**
- пользователь выбирает действие «Удалить из комнаты»
- подтверждает действие

**Тогда**
- устройство удаляется из комнаты
- пользователь видит сообщение:  
  Устройство удалено из комнаты.
- устройство отображается в общем списке устройств дома

### Сценарий 2: Если удаление невозможно, отображается ошибка

**Дано**
- авторизованный пользователь находится на экране комнаты
- в списке устройств выбрано нужное устройство

**Когда**
- пользователь выбирает действие «Удалить из комнаты»
- подтверждает действие
- удаление невозможно

**Тогда**
- устройство остаётся в комнате
- отображается ошибка:  
  Не удается удалить устройство из комнаты -- попробуйте еще раз позже.
