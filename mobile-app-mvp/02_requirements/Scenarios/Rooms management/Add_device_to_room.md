## Добавление устройства в комнату

**Как пользователь**,  
я хочу добавлять устройство в комнату,  
**чтобы** управлять им вместе с другими устройствами этого помещения.

### Примечания
- Устройство должно быть предварительно добавлено в приложение.
- Одно устройство может быть привязано только к одной комнате.

### Критерии приёмки
- Авторизованный пользователь может добавить устройство в комнату.
- Если устройство уже добавлено в другую комнату, отображается ошибка.
- Если пользователь не выбрал устройство, отображается ошибка.

### Сценарий 1: Авторизованный пользователь может добавить устройство в комнату

**Дано**
- авторизованный пользователь находится в приложении
- в доме есть устройства, не привязанные к комнатам

**Когда**
- пользователь выбирает комнату
- выбирает опцию «Добавить устройство»
- выбирает устройство из списка
- подтверждает добавление

**Тогда**
- устройство добавляется в выбранную комнату
- пользователь видит сообщение:
> Устройство добавлено в комнату.

### Сценарий 2: Если устройство уже добавлено в другую комнату, отображается ошибка

**Дано**
- устройство уже привязано к другой комнате
- авторизованный пользователь находится в приложении

**Когда**
- пользователь пытается добавить это устройство в текущую комнату

**Тогда**
- пользователь видит сообщение:
> Устройство уже добавлено в другую комнату.

- устройство не добавляется

### Сценарий 3: Если пользователь не выбрал устройство, отображается ошибка

**Дано**  
авторизованный пользователь находится на экране добавления устройства в комнату

**Когда**
- пользователь не выбирает устройство
- подтверждает добавление

**Тогда**
- пользователь видит сообщение:
> Выберите устройство для добавления.

- устройство не добавляется
