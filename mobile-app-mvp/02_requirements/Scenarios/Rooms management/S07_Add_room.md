## Добавление комнаты в дом

**Как пользователь**,  
я хочу иметь возможность добавить комнату в умный дом и выбрать её тип,  
**чтобы** можно было управлять умными устройствами в разных комнатах.

### Примечания
- В одном доме может быть максимум **10 комнат**.
- У каждой комнаты в доме **уникальное название и иконка**.
- Необходимо отрисовать **иконки для всех типов комнат** в интерфейсе.
- Название комнаты может совпадать с её типом.
- Комната не может существовать отдельно от дома.
- После создания дома в нём нет комнат.

### Критерии приёмки
- Авторизованный пользователь может добавить новую комнату и выбрать её тип.
- Если пользователь указывает название, которое уже используется, отображается ошибка.
- Если пользователь пытается создать комнату без названия, отображается ошибка.

### Сценарий 1: Авторизованный пользователь может добавить новую комнату и выбрать её тип

**Дано**  
авторизованный пользователь находится в приложении

**Когда**
- выбирает опцию «Добавить комнату»
- вводит `<название комнаты>`
- выбирает `<тип комнаты>` из заранее заданного списка
- подтверждает добавление

**Тогда**
- комната добавляется в список комнат дома
- пользователь видит сообщение:
Комната успешно добавлена.


#### Примеры

| Тип комнаты | Название комнаты |
|------------|------------------|
| Спальня    | Комната Жеки     |
| Спальня    | Комната Серёги   |
| Кухня      | Кухня            |
| Другое     | Винный погреб    |

### Сценарий 2: Если пользователь указывает название комнаты, которое уже используется, отображается ошибка

**Дано**  
авторизованный пользователь находится на экране добавления комнаты

**Когда**
- вводит название, которое уже есть в списке комнат
- подтверждает добавление комнаты

**Тогда**
- пользователь видит сообщение об ошибке:
> Комната с таким названием уже существует.

- комната не добавляется

### Сценарий 3: Если пользователь пытается создать комнату без названия, отображается ошибка

**Дано**  
авторизованный пользователь находится на экране добавления комнаты

**Когда**
- оставляет поле ввода названия пустым
- подтверждает добавление комнаты

**Тогда**
- пользователь видит сообщение об ошибке:
> Введите название комнаты

- комната не добавляется
