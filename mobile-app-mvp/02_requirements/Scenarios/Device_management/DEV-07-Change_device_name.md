# Изменение названия устройства

**Когда:** как пользователь, я хочу изменять название устройства, чтобы удобно отличать их друг от друга.  

**Примечание:** по умолчанию название устройства = название модели, ограничений на длину нет.  

**Критерии приёмки:**  
- Пользователь может изменить название устройства.  
- Если устройство с таким названием уже есть, отображается ошибка.  

**Сценарий 1: Успешное изменение**  
- **Дано:** авторизованный пользователь добавил хотя бы одно устройство  
- **Когда:** выбирает устройство → действие «Изменить название» → вводит новое название → подтверждает  
- **Тогда:**  
  - Название меняется  
  - Пользователь видит сообщение:  
    ```
    Имя устройства успешно изменено
    ```

**Примеры:**  

| Текущее название | Новое название |
|-----------------|----------------|
| LampStets2034   | Лампочка Ильича |
| Лампа1          | Lamp           |

**Сценарий 2: Ошибка — название уже существует**  
- **Дано:** авторизованный пользователь добавил хотя бы одно устройство  
- **Когда:** вводит новое название, которое уже есть у другого устройства → подтверждает  
- **Тогда:**  
  - Сообщение:  
    ```
    Устройство с таким именем уже есть
    ```  
  - Название не меняется
