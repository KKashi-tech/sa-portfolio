# Сценарий 4: Удаление лампы Stets из приложения

**Когда:** как пользователь, я хочу иметь возможность удалить лампу Stets, чтобы в умном доме не было неиспользуемых или неисправных устройств.  

**Примечание:** удалённое устройство можно снова добавить в приложение.  

**Критерии приёмки:**  
- Авторизованный пользователь может удалить лампу Stets.  
- Если удаление не удалось, отображается сообщение об ошибке.  

**Сценарий 1: Успешное удаление**  
- **Дано:** авторизованный пользователь заходит в настройки устройства  
- **Когда:** выбирает опцию удаления  
- **Тогда:**  
  - Устройство удаляется из приложения и списка  
  - Может быть добавлено снова  
  - Пользователь видит сообщение:  
    ```
    Устройство успешно удалено -- вы можете добавить его снова
    ```

**Сценарий 2: Ошибка при удалении**  
- **Дано:** авторизованный пользователь заходит в настройки устройства  
- **Когда:** выбирает опцию удаления, устройство не может быть удалено  
- **Тогда:**  
  - Сообщение:  
    ```
    При удалении устройства возникла ошибка. Попробуйте еще раз.
    ```
