# Сценарий 9: Ручной запуск сценария

**Когда:** как пользователь, я хочу запускать сценарий с ручным запуском одной кнопкой.  

**Критерии приёмки:**  
- Сценарий может быть запущен вручную  
- Сообщение о недоступных устройствах или ошибке при запуске  

**Сценарий 1: Успешный запуск вручную**  
- **Дано:** сценарий с ручным запуском в списке  
- **Когда:** пользователь выбирает сценарий и действие `<действие 2>`  
- **Тогда:**  
  - Статус → «Выполняется сейчас»  
  - Все устройства выполняют действия  
  - Сообщение:  
    ```
    <Сценарий> запущен
    ```  
  - По окончании статус → «Выключен»  

**Примеры:**  

| Сценарий        | Статус             | Запуск по времени | Действие 1              | Действие 2         |
|-----------------|------------------|-----------------|------------------------|------------------|
| Готовлюсь ко сну | Выключен          | Не запускается  | Включить выполнение    | Запустить вручную |
| Утренний свет   | Выполняется сейчас| Не запускается  | Выключить выполнение   | Выключить        |
| Рабочий свет    | Выключен          | Не запускается  | Включить выполнение    | Запустить вручную |

**Сценарий 2: Недоступные устройства**  
- **Дано:** сценарий с ручным запуском  
- **Когда:** некоторые устройства недоступны  
- **Тогда:**  
  - Сценарий запускается  
  - Сообщение:  
    ```
    Некоторые устройства из сценария недоступны или используются другим сценарием: Lamp1, Лампа 2
    ```

**Сценарий 3: Все устройства недоступны**  
- **Дано:** сценарий с ручным запуском  
- **Когда:** все устройства недоступны  
- **Тогда:** ошибка:  
> Не удалось запустить сценарий - проверьте доступность устройств или подключение к интернету и повторите еще раз
