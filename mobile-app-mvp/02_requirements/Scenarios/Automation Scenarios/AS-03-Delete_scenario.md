# Сценарий 8: Удаление сценариев автоматизации

**Когда:** как пользователь, я хочу удалять сценарии, чтобы не держать ненужные в списке.  

**Критерии приёмки:**  
- Сценарий можно удалить  
- При любой ошибке отображается сообщение  

**Сценарий 1: Успешное удаление**  
- **Дано:** авторизованный пользователь в разделе сценариев  
- **Когда:** выбирает сценарий → «Удалить сценарий» → подтверждает  
- **Тогда:**  
  - Сценарий удалён  
  - Сообщение:  
    ```
    Сценарий "<Название сценария>" удалён.
    ```

**Сценарий 2: Ошибка — сценарий выполняется**  
- **Дано:** сценарий в активном состоянии  
- **Когда:** пользователь выбирает «Удалить»  
- **Тогда:**  
  - Сообщение:  
    ```
    Невозможно удалить сценарий во время выполнения.
    ```  
  - Сценарий не удаляется

**Сценарий 3: Ошибка при удалении**  
- **Дано:** пользователь в списке сценариев  
- **Когда:** происходит ошибка при удалении  
- **Тогда:**  
  - Сообщение:  
    ```
    Ошибка при удалении сценария. Попробуйте еще раз позже.
    ```  
  - Сценарий не удаляется
