# Выход из приложения

**Когда:** как пользователь, я хочу выходить из приложения, чтобы другие не получили доступ к моим данным.  

**Критерии приёмки:**  
- Пользователь может выйти вручную  
- При ошибке выхода отображается сообщение  
- При повторном открытии приложения пользователь не авторизован  

**Сценарий 1: Успешный выход**  
- **Дано:** авторизованный пользователь открыл приложение  
- **Когда:** выбирает «Выйти из учетной записи»  
- **Тогда:**  
  - Завершение сессии  
  - Пользователь попадает на приветственный экран  
  - Видит опции «Войти» и «Зарегистрироваться»  

**Сценарий 2: Ошибка при выходе**  
- **Дано:** пользователь на главном экране  
- **Когда:** выбирает «Выйти», возникает ошибка  
- **Тогда:**  
  - Сообщение:  
    ```
    Произошла ошибка при выходе. Попробуйте позже.
    ```  
  - Пользователь остаётся на текущем экране  

**Сценарий 3: После повторного открытия**  
- **Дано:** пользователь вышел и закрыл приложение  
- **Когда:** открывает приложение  
- **Тогда:** приветственный экран, автоматического входа нет
