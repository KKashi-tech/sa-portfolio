# S06 - Добавление пользователя в дом

**Как пользователь**, я хочу добавить другого пользователя в дом, чтобы он тоже мог управлять умными устройствами.

### Критерии приёмки
- Можно добавить только одного дополнительного пользователя.
- Пользователь должен быть зарегистрирован.
- Повторное добавление — ошибка.

### Сценарий 1: Успешное добавление

**Дано**
- авторизованный пользователь на экране настроек дома  

**Когда**
- выбирает «Добавить другого человека»
- вводит корректный email
- подтверждает действие

**Тогда**
- пользователь добавляется в дом
- отображается сообщение:  
  > Пользователь успешно добавлен в дом.

---

### Сценарий 2: Некорректная почта

**Тогда**
> Не удалось добавить участника. Проверьте правильность адреса электронной почты.

---

### Сценарий 3: Превышен лимит

**Тогда**
> Невозможно добавить человека: в одном доме может быть только два пользователя.

---

### Сценарий 4: Пользователь уже добавлен

**Тогда**
> Этот пользователь уже является участником дома.
