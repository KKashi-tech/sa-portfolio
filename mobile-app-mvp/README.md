## Обзор проекта
Данный проект представляет собой кейс системного анализа для мобильного приложения «Умный дом».  
Ключевые цели проекта: структурировать бизнес- и пользовательские потребности, определить объём и приоритеты MVP, перевести требования в формализованные системные артефакты и подготовить материалы для этапов разработки, дизайна и тестирования.

## Моя роль
- Анализ и формализация потребностей пользователей и бизнеса 
- Определение скоупа релиза 1.0 и формирование MVP [01_MVP_scope]
- Составление спецификации к приложению для передачи в разработку: описание функциональных и нефункциональных требований, формирование критериев приемки и пользовательских сценариев [02_REQUIREMENTS]
- Создание прототипа UI приложения для обсуждения с заказчиком и передачи дизайнерам [03_UI]
- Составление тест-плана для команды тестирования [04_testing_plan]
